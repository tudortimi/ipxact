The TLM kit is required for compiling and running the example.

The TLM 1.0 kit can be found at:
http://www.systemc.org/downloads/standards/

After installing TLM 1.0, all files in the kit are referenced from the root of the installation directory represented as $TLM_HOME.
Make sure you have defined this environment variable correctly.

The include dir of the SystemC kit is referenced as $SYSTEMC_HOME
